## Quick Visual Customization

### Change Accent Color (Currently: #F4A460)
Find in style.css:
```css
--accent-color: #F4A460;  ← Change this value
```

Examples:
- Red: #E74C3C
- Green: #27AE60
- Blue: #3498DB
- Purple: #9B59B6

### Change Font Size
Find in style.css:
```css
h2 { font-size: 2.5rem; }  ← Larger = bigger heading
```

### Change Button Radius
Find in style.css:
```css
--border-radius: 12px;  ← Larger = rounder corners
```

### Change Spacing
Find in style.css:
```css
gap: 30px;  ← Larger = more space between items
```
