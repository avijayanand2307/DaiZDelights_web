DATA FLOW:

1. User visits index.html (Home)
2. User navigates to products.html
3. User clicks Order Now → order-now.html
4. User fills delivery information
5. User selects products (checkboxes)
6. Order Summary updates in real-time
7. User optionally applies promo code
8. User submits form
9. Data saved to browser localStorage
10. Confirmation notification shown
